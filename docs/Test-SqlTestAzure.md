# Test-SqlTestAzure

## SYNOPSIS
Tests if a Azure-based SQL Server can be created.

## SYNTAX

```
Test-SqlTestAzure [<CommonParameters>]
```

## DESCRIPTION
Checks if Az is installed, and if the Azure service is running.

## EXAMPLES

### EXAMPLE 1
```
Test-SqlTestAzure
```
