# Test-SqlTestLocalDb

## SYNOPSIS
Tests if a localDb-based SQL Server can be created.

## SYNTAX

```
Test-SqlTestLocalDb [<CommonParameters>]
```

## DESCRIPTION
Uses \[SqlLocalDB Utility\](https://docs.microsoft.com/en-us/sql/tools/sqllocaldb-utility?view=sql-server-ver15).

## EXAMPLES

### EXAMPLE 1
```
Test-LocalDb
```

True
